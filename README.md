# Core Java Practice Repository ☕🚀

This repository contains a collection of Java programs I wrote while learning and practicing Core Java and Object-Oriented Programming (OOP) concepts.

## 📚 Concepts Covered

### 1. Object-Oriented Programming (OOP)
* **Encapsulation:** (`Encapsulations.java`)
* **Inheritance:** (`Inheritance.java`)
* **Constructors:** (`Constructorr.java`, `Cons.java`)

### 2. Data Structures & Basics
* **Arrays:** Single and Multi-dimensional arrays (`Array.java`, `Multiarray.java`, `Jagged.java`)
* **Keywords:** Understanding access modifiers and static variables (`Staticc.java`, `pubic.java`)

### 3. Practice Exercises
* Basic Calculator implementations (`Calculator.java`, `Calculatorr.java`)
* General programming exercises (`Exercise.java`, `Hello.java`)

## 🛠️ How to Run
To run any of these files on your local machine:
1. Open your terminal in this directory.
2. Compile the specific file: `javac FileName.java` (e.g., `javac Array.java`)
3. Execute the compiled class: `java FileName` (e.g., `java Array`)
